Создание фикстур для подготовки таблиы предсказаний

Предсказания сохраняются с WEB страницы в текстовый файл.
При копировании строки могут быть пронумерованы и иметь переносы в неположенных местах.

Для формирования файла с предсказаниями в котором каждое предсказание записанов отдельной строке
используется edit_text_file.py, который создает файл predictions_new.txt

Для импорта данных в модель Django нужно создать файл фиктсуры в формате JSON.
Для создания файла fixture_prediction.json используется to_fixture.py,
который читает predictions_new.txt

Затем файл fixture_prediction.json копируем в папку fixtures приложения happy_new_year
и фыполняем импорт командой:
python manage.py loaddata fixture_prediction.json
